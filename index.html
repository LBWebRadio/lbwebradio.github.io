Version 0.8.1.0 Released

October 27, 2025

Enhanced error recovery and network stability update. Redesigned error handler now offers Restart, View Debug Info, Continue or Exit options for unexpected errors. Added network connection loss protection with graceful streaming shutdown and user notifications. Improved random station selection avoids both current and previous stations. Enhanced metadata filtering excludes error messages. Added database maintenance option to delete records older than 12 months. Fixed duplicate shutdown loop and UI dialog stacking issues. Updated WebView2 to latest version.

Version 0.8.0.0 Released

October 19, 2025

Major database and stability update. Improved database checkpoint system with automatic retry logic and better handling of locks and busy states. Enhanced shutdown process now properly saves all changes with retry before closing, significantly reducing risk of data loss. Database export includes all recent changes even if not yet saved, and import operations now prevent corruption from background processes. Fixed critical crash when closing Advanced Radio Browser during shutdown and rare null reference crashes. Random station button now avoids consecutive repeats, and "Time Played" renamed to "Streaming".
