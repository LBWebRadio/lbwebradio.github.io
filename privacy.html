Privacy Policy

Live Beats WebRadio · Last updated: July 1, 2026

Live Beats WebRadio ("the app"), identified by the package name com.lbwebradio.radio, is an internet radio player developed by Christian Flach. This policy explains what data is stored on your device and what limited technical data is shared with third-party services required for the app to function.

Summary

  • No account required. No personal data collected.
  • No ads, analytics, or tracking SDKs.
  • All user data stays on your device.
  • Only essential technical requests are made for streaming and station lookup.

Data Stored on Your Device

  • Favorite stations and custom tags
  • App settings (theme, language, sleep timer)
  • Listening history and usage statistics
  • Saved songs
  • Export backups (.lbrx files)

Technical Data Shared with Third Parties

Service Purpose Data
Radio-Browser API Station search and directory Search queries, filters, votes, play counts
Streaming Servers Audio playback Direct stream URL connection
ipwho.is API Server location lookup IP address of radio server
Apple iTunes / Last.fm Album artwork lookup Artist and track name

You can disable Album Art and Server Location in settings to reduce external requests.

Legal Basis (GDPR)

We process minimal technical data under legitimate interest to provide core functionality such as streaming and station directory services.

Security & Data Deletion

  • HTTPS is used where available
  • Some radio streams may be HTTP (station dependent)
  • All data is stored locally and deleted on uninstall

Children’s Privacy

This app is not directed at children under 13 and does not knowingly collect personal data.
