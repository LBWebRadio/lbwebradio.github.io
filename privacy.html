Privacy Policy

Live Beats WebRadio · Last updated: July 1, 2026

Summary

  • No account required — no personal data collected
  • No ads, analytics, or tracking SDKs
  • All user data stays on your device
  • Only essential technical requests are made for streaming

Data Stored on Your Device

  • Favorites and custom station tags
  • App settings (theme, language, sleep timer)
  • Listening history and usage stats
  • Saved song titles
  • Exported backup files (.lbrx)

Third-Party Services

Service Purpose Data
Radio-Browser Station search Search queries, filters, votes, play counts
Streaming servers Audio playback Stream URL connection
ipwho.is Server location lookup IP of radio server
iTunes / Last.fm Album artwork Artist + track name

You can disable album art and server location in settings to reduce external requests.

Legal Basis (GDPR)

We process minimal technical data under legitimate interest to provide streaming and station directory functionality.

Security & Deletion

  • HTTPS is used where available
  • Some streams may be HTTP (radio station limitation)
  • All data is stored locally and deleted on uninstall
